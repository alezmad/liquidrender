### Design Tenets

1. **60-Second Value** - User sees real insights from their data within 60 seconds
2. **AI Does The Work** - System infers, user confirms (not the reverse)
3. **Progressive Disclosure** - Show complexity only when needed
4. **Escape Hatches Everywhere** - Never trap users in flows
5. **Refinement Over Setup** - Onboarding transitions to ongoing refinement
6. **Transparency Builds Trust** - Show confidence levels, explain reasoning

### Interaction Model

| Traditional BI | Knosia |
|----------------|--------|
| Configure â†’ Use | Use â†’ Refine |
| Forms collect data | Decisions confirm inference |
| Dashboard is empty canvas | Dashboard is generated product |
| User builds queries | User asks questions |
| Static reports | Living intelligence |

## Appendix: Component Library Reference

### Design Tokens

| Token | Value | Usage |
|-------|-------|-------|
| `--color-success` | `#22c55e` | Positive trends, confirmations |
| `--color-warning` | `#f59e0b` | Anomalies, attention needed |
| `--color-error` | `#ef4444` | Errors, negative trends |
| `--color-muted` | `#6b7280` | Secondary text, disabled |
| `--radius-sm` | `4px` | Buttons, inputs |
| `--radius-md` | `8px` | Cards, panels |
| `--spacing-unit` | `4px` | Base spacing multiplier |

### Typography Scale

| Name | Size | Weight | Usage |
|------|------|--------|-------|
| `heading-1` | 32px | 700 | Page titles |
| `heading-2` | 24px | 600 | Section headers |
| `heading-3` | 18px | 600 | Card titles |
| `body` | 14px | 400 | Default text |
| `caption` | 12px | 400 | Labels, metadata |
| `metric` | 28px | 700 | KPI values |

### Animation Principles

- **Duration**: 150-300ms for micro-interactions
- **Easing**: `ease-out` for entrances, `ease-in` for exits
- **Reduced motion**: Respect `prefers-reduced-motion`
- **Loading**: Skeleton screens over spinners when possible
