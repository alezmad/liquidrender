## Knosia — B2B Business Intelligence

### Brand Block

```
┌─────────────────────────────────────────┐
│                                         │
│              Knosia                     │
│                                         │
│         Know what matters.              │
│                                         │
└─────────────────────────────────────────┘
```

### Name Etymology

- **"Know"** — English, the core action
- **"Gnosis"** — Greek (γνῶσις), knowledge/insight
- **Pronunciation:** NO-see-ah
- **Letters:** 6
- **Syllables:** 3

### Domain

- **Primary:** knosia.com
- **Available:** knosia.ai, knosia.io

### Slogan

```
Know what matters.
```

### Tagline Extensions

```
Know what matters.
Your business, briefed daily.
```

```
Know what matters.
Intelligence, delivered.
```

```
Know what matters.
The rest is noise.
```

### The Promise

We don't give you data. We tell you what's important.

| Not This | This |
|----------|------|
| Dashboards | Briefings |
| Queries | Answers |
| Charts | Insights |
| Numbers | Stories |

### Positioning Statement

> Knosia is for business leaders who are drowning in data but starving for insight. Unlike dashboards that wait for questions, Knosia delivers daily briefings that tell you what matters—before you ask.

### The Anti-Positioning

**We are NOT:**
- A dashboard builder
- A BI tool
- A query interface
- A visualization platform

**We ARE:**
- Your daily briefing
- An intelligence layer
- A business translator
- The data scientist you can't afford

### Emotional Core

| Before | After |
|--------|-------|
| "I have data everywhere but no answers" | "I wake up knowing what needs my attention" |
| "I'm flying blind" | "I know what matters" |
| "Waiting for the data team" | "Already briefed" |

### Visual Identity Direction

- Clean, minimal (clarity, not complexity)
- Morning light aesthetic (the briefing arrives at dawn)
- Confident typography (you know, not guess)
- No chart clutter (signal, not noise)

## Intent Is Enough — B2C Agentic Workflows

### Brand Block

```
┌─────────────────────────────────────────┐
│                                         │
│         Intent Is Enough                │
│                                         │
│      Describe it. We'll do it.          │
│                                         │
└─────────────────────────────────────────┘
```

### Domain

- **Primary:** intentisenough.com

### Core Concept

Users describe what they want → AI agents execute autonomously.

No tools. No config. No waiting.

### Use Cases

1. **Apartment hunting** — Find apartments matching criteria
2. **Sales prospecting** — Identify leads, generate outreach
3. **Supply chain** — Find alternative suppliers

### Positioning

| Competitor | Limitation |
|------------|------------|
| ChatGPT | Text output, manual copy-paste |
| Zapier/n8n | Pre-built workflows, self-config |
| Vertical SaaS | Rigid interfaces |

**Intent Is Enough:** Workflows generated from intent, UI adapts dynamically.

## LiquidRender — Technology Layer

### Domain

- **Primary:** liquidrender.com

### Purpose

The rendering/intelligence engine powering both Knosia (B2B) and Intent Is Enough (B2C).

Developer-facing. API-first. The infrastructure.
